# Unherd

A collaborative recommender system that mitigates herding effects
due to popularity and conformity bias. Developed as a course project
for AI for Humanity.

Old code is in [this notebook](AI_for_Humanity.ipynb). This is gradually
being refactored into individual Python files. For our report, see
[Team_9_Report.pdf](Team_9_Report.pdf).

TODO: Web UI.

## Usage

Clone the repo:
```
git clone https://github.com/shivrm/unherd
cd unherd
```

Train the recommender models:
```
mkdir models
python src/train.py
```
